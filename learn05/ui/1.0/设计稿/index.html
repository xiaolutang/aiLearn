📋 设计概述

本设计稿集合展示了智能教学助手应用的完整用户界面设计，涵盖了从用户登录到核心功能使用的全流程体验。设计遵循现代化的UI/UX设计原则，注重用户体验的流畅性和功能的易用性，特别针对教育场景进行了深度优化。

设计采用了一致的视觉语言和交互模式，确保用户在不同功能模块间切换时能够保持良好的使用体验。色彩方案以专业的蓝色系为主，辅以活力橙色作为点缀，营造出既专业又亲和的视觉氛围。

🎨 设计规范

  • 主色调 #1890FF
  • 辅助色 #FA8C16
  • 字体 PingFang SC
  • 圆角 6px - 16px
  • 间距 8px 倍数

📱 响应式断点

  • 桌面端 > 1024px
  • 平板端 768px - 1024px
  • 移动端 < 768px
  • 最小宽度 320px
