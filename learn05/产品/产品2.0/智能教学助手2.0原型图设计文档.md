# 智能教学助手2.0 原型图设计文档

## 文档概述

### 设计目标
- 为智能教学助手2.0提供完整的界面原型设计
- 建立统一的设计语言和交互规范
- 指导UI设计师进行高保真界面设计
- 确保用户体验的一致性和易用性

### 设计原则
- **简洁直观**: 界面简洁明了，操作直观易懂
- **智能高效**: 充分体现AI能力，提升工作效率
- **响应式设计**: 适配不同设备和屏幕尺寸
- **无障碍设计**: 考虑特殊用户群体的使用需求

### 目标用户
- **主要用户**: K12教师（学科教师、班主任）
- **次要用户**: 学校管理员、教研组长
- **间接用户**: 学生、家长

## 设计系统规范

### 色彩系统
```
主色调 (Primary Colors):
- 主蓝色: #1890FF (品牌色，用于主要按钮、链接)
- 深蓝色: #0050B3 (悬停状态、强调)
- 浅蓝色: #E6F7FF (背景、辅助区域)

辅助色 (Secondary Colors):
- 成功绿: #52C41A (成功状态、正向反馈)
- 警告橙: #FA8C16 (警告提示、注意事项)
- 错误红: #F5222D (错误状态、删除操作)
- 信息蓝: #13C2C2 (信息提示、中性反馈)

中性色 (Neutral Colors):
- 标题黑: #262626 (主要文字)
- 正文灰: #595959 (正文内容)
- 辅助灰: #8C8C8C (辅助信息)
- 禁用灰: #BFBFBF (禁用状态)
- 边框灰: #D9D9D9 (分割线、边框)
- 背景灰: #FAFAFA (页面背景)
- 纯白色: #FFFFFF (卡片背景)
```

### 字体系统
```
字体族:
- 中文: PingFang SC, Microsoft YaHei, 微软雅黑
- 英文: -apple-system, BlinkMacSystemFont, Segoe UI, Roboto
- 数字: Helvetica Neue, Arial

字体大小:
- 大标题: 24px (页面主标题)
- 中标题: 20px (模块标题)
- 小标题: 16px (卡片标题)
- 正文: 14px (主要内容)
- 辅助文字: 12px (说明文字)
- 小字: 10px (标签、状态)

字重:
- 粗体: 600 (标题、重要信息)
- 常规: 400 (正文内容)
- 细体: 300 (辅助信息)
```

### 间距系统
```
基础间距单位: 8px

常用间距:
- 4px: 紧密间距 (图标与文字)
- 8px: 小间距 (表单元素内部)
- 16px: 中间距 (卡片内容)
- 24px: 大间距 (模块之间)
- 32px: 超大间距 (页面区块)
- 48px: 页面边距
```

## 响应式设计规范

### 断点设置
```
超小屏幕 (xs): < 576px  (手机竖屏)
小屏幕 (sm): 576px - 768px  (手机横屏)
中等屏幕 (md): 768px - 992px  (平板)
大屏幕 (lg): 992px - 1200px  (小型桌面)
超大屏幕 (xl): > 1200px  (大型桌面)
```

### 布局适配策略

#### 移动端适配 (xs, sm)
- 侧边栏收起为底部导航栏
- 卡片布局改为单列显示
- 表格支持横向滚动
- 按钮尺寸增大便于触摸

#### 平板端适配 (md)
- 侧边栏可收起/展开
- 卡片布局2列显示
- 保持桌面端交互方式

#### 桌面端 (lg, xl)
- 完整功能展示
- 多列布局充分利用空间
- 支持快捷键操作

## 交互动效规范

### 基础动效
```css
/* 标准过渡 */
.transition-standard {
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

/* 快速过渡 */
.transition-fast {
  transition: all 0.15s cubic-bezier(0.4, 0, 0.2, 1);
}

/* 慢速过渡 */
.transition-slow {
  transition: all 0.5s cubic-bezier(0.4, 0, 0.2, 1);
}
```

### 悬停效果
- 按钮: 背景色变化 + 轻微阴影
- 卡片: 阴影加深 + 轻微上移
- 链接: 颜色变化 + 下划线

### 加载动效
- 骨架屏: 内容加载时显示
- 进度条: 长时间操作反馈
- 旋转图标: 短时间加载提示

## 无障碍设计规范

### 键盘导航
- 支持Tab键顺序导航
- 焦点状态清晰可见
- 支持快捷键操作

### 屏幕阅读器支持
- 语义化HTML标签
- 适当的ARIA标签
- 图片添加alt描述

### 色彩对比
- 文字与背景对比度 ≥ 4.5:1
- 重要信息对比度 ≥ 7:1
- 支持高对比度模式

## 性能优化指导

### 图片优化
- 使用WebP格式
- 实现懒加载
- 提供多尺寸适配

### 代码分割
- 按模块分割代码
- 路由级别懒加载
- 第三方库按需引入

### 缓存策略
- 静态资源长期缓存
- API数据适当缓存
- 离线功能支持
